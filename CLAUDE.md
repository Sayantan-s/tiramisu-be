## Overview

`tiramisu-be` is a Python FastAPI backend. Dependencies and the virtualenv are managed with [uv](https://docs.astral.sh/uv/) (see `uv.lock`). Python >=3.11 is required (pinned to 3.11 via `.python-version`).

## Commands

- Run the dev server (auto-reload): `uv run fastapi dev`
- Run the app directly: `uv run python main.py` (serves on `127.0.0.1:8000` with reload)
- Add a dependency: `uv add <package>` — do not edit `pyproject.toml` deps by hand; uv updates `uv.lock`
- Sync the environment from the lockfile: `uv sync`

There is no test suite or linter configured yet. When adding tests, prefer `uv add --dev pytest` and run with `uv run pytest` (single test: `uv run pytest path::test_name`).

## Architecture

The entire app currently lives in `main.py`: a single `FastAPI()` instance named `app` with route handlers defined inline. The `app` symbol is the ASGI entrypoint referenced as `main:app` by uvicorn/`fastapi dev`.

Interactive API docs are auto-generated by FastAPI at `/docs` (Swagger) and `/redoc`.

As routes grow, the conventional next step is to split handlers into `APIRouter` modules and `app.include_router()` them in `main.py`, keeping `main.py` as the assembly point.
